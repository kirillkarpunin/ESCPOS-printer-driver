# C++20 ESC/POS Lib

Библиотека для работы с чековыми термопринтерами по протоколу ESC/POS.

## Особенности

- Печать текста (с возможностью настройки размера, шрифта, стилей, отступов и проч.)
- Печать изображений

## Описание классов

#### `Connection`
Класс, описывающий подключение к принтеру. Реализует метод записи в файловый дескриптор, используемый для связи с принетром. Имеет двух наследников - `ComPortConnection` и `BluetoothConnection`. 

#### `ComPortConnection`
Класс, представляющий собой проводное подключение к принтеру. Имеет единственный конструктор, принимающий строку - путь к файлу принтера в системе.

#### `BluetoothConnection`
Класс, представляющий собой подключение к принтеру через Bluetooth. Имеет единственный конструктор, принимающий строку - MAC-адрес принтера.
