# C++20 ESC/POS Lib

Библиотека для работы с чековыми термопринтерами по протоколу ESC/POS.

## Особенности

- Печать текста (с возможностью настройки размера, шрифта, стилей, отступов и проч.)
- Печать изображений

## Описание классов

#### `Connection`
Класс, описывающий подключение к принтеру. Реализует метод записи в файловый дескриптор, используемый для связи с принетром. Имеет двух наследников - `ComPortConnection` и `BluetoothConnection`. 

#### `ComPortConnection`
Класс, представляющий собой проводное подключение к принтеру. Имеет конструктор, принимающий строку - путь к файлу принтера в системе.

#### `BluetoothConnection`
Класс, представляющий собой подключение к принтеру через Bluetooth. Имеет конструктор, принимающий строку - MAC-адрес принтера.

---

#### `Printer`
Главный класс библиотеки, предоставляющий интерфейс для управления термопринтером. Имеет конструктор, принимающий объекты наследников класса `Connection`.

**Методы:**
- `void initialize() const` - Метод инициализации принтера, который необходимо вызвать перед работой с принтером. Очищает буфер печати и отменяет текущее форматирование.
<br>

- `void write(const std::string& data) const`
- `void write(const unsigned char& ch) const` - Добавляют строку или одиночный символ в буфер печати принтера без команды переноса строки.
- `void print() const` - Отправляет команду переноса строки, заставляя принтер напечатать все символы, находящиеся в буфере печати.
- `void print(const std::string& data) const`
- `void print(const unsigned char& ch) const` - Добавляют строку или одиночный символ в буфер печати принтера и отправляет команду переноса строки для немедленной печати.
<br>

- `void set_text_feature(Text::TextFeature feature, bool v)` - Включает (`true`) или выключает (`false`) стиль текста (полужирный, курсив, инверсия, подчеркивание).
- `void reset_text_features()` - Выключает все включенные стили текста.
- `bool get_text_feature(Text::TextFeature feature) const` - Возвращает текущее состояние конкретного стиля текста.
- `void set_text_underline_thickness(Text::UnderlineThickness v)` - Задает толщину линии подчеркивания - стандартная (`x1`) или двойная (`x2`).
- `Text::UnderlineThickness get_text_underline_thickness() const` - Возвращает текущую толщину линии подчеркивания.
- `void set_font(Text::Font v)` - Задает шрифт текста - стандартный (`A`) или альтернативный (`B`).
<br>

- `void set_text_width_scale(Text::Scale v)`
- `void set_text_height_scale(Text::Scale v)`
- `void set_text_scale(Text::Scale width, Text::Scale height)` - Задают множитель размера символов текста по ширине и/или высоте соответственно (от `x1` до `x4`).
- `void reset_text_scale()` - Сбрасывает множители размера по ширине и высоте к стандартному размеру (`x1`).
- `Text::Scale get_text_width_scale() const`
- `Text::Scale get_text_height_scale() const` - Возвращают текущие множители размера по ширине и высоте соответственно.
- `void set_rotation(Text::Angle v)` - Устанавливает поворот печатаемого текста на `0`, `90`, `180` или `270` градусов.
- `void reset_rotation()` - Сбрасывает установленное значение поворота печатаемого текста к стандартному (`0` градусов).
<br>

- `void justify(Justification v)` - Устанавливает позицию печати текста: по левому краю, по центру или по правому краю.
- `void set_absolute_print_position(uint16_t v)` - Задает позицию начала печати текста (в точках) от левого края чековой ленты.
- `void set_right_side_character_spacing(uint8_t v)` - Задает промежуток (в точках) справа от каждого символа печатаемого текста.
- `void set_line_spacing(uint8_t v)` - Задает высоту межстрочного интервала (в точках).
- `void reset_line_spacing()` - Сбрасывает значение высоты межстрочного интервала к стандартному.
- `void set_left_margin(uint16_t v)` - Задает отступ левого поля (в точках) для всей области печати.
- `void set_print_area_width(uint16_t v)` - Задает ограничение ширины области печати (в точках).
<br>

#### `Text`
Структура для хранения текущих настроек печатаемого текста. Экземпляр этой структуры содержится в классе `Printer`.

Описывает ряд `enum`'ов, ожидаемых в методах класса `Printer`:
- `TextFeature` - имеет значения `BOLD`, `ITALIC`, `INVERTED`, `UNDERLINED`.
